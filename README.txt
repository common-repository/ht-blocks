== Description ==

HT Blocks is a [Gutenberg](https://wordpress.org/gutenberg/) Blocks plugin which will provide you a collection of beautiful, customizable Gutenberg blocks for the new block editor.

You will need Gutenberg installed for this plugin to work.

## HT Blocks currently includes the following blocks:

* Accordion | [Block Demo](http://demo.wphash.com/htblocks/accordions/)
* Progress Bar (3 Style) | [Block Demo](http://demo.wphash.com/htblocks/progress-brs/)
* Countdowns (7 Style) | [Block Demo](http://demo.wphash.com/htblocks/countdowns/)
* Pricing Table (7 Style) | [Block Demo](http://demo.wphash.com/htblocks/pricing-table/)
* Team Member(6 Style) | [Block Demo](http://demo.wphash.com/htblocks/team-member/)
* Testimonial (3 Style) | [Block Demo](http://demo.wphash.com/htblocks/testimonial/)

We plan to add more in the future.

Each blocks are customizable:
All of the blocks are customizable with lots of options.

== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to _Plugins &rarr; Add New_.
2. Type "HT Blocks" into the Search and hit Enter.
3. Locate the HT Blocks plugin in the list of search results and click **Install Now**.
4. Once installed, click the Activate link.
5. You will get the blocks in the Gutenberg editor.

It can also be installed manually.

1. Download the HT Blocks plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Locate HT Blocks in the list and click the *Activate* link.
